## **My Answer**

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        par = {")" : "(", "}" : "{", "]" : "["}

        for i in s:
            if i in par.values():
                stack.append(i)
            elif i in par.keys():
                if not stack or stack[-1] != par[i]:
                    return False
                stack.pop()
        return len(stack)==0
```

### **Solution**  

#### **Approach**
Using stack, compare each value with the top of stack. If the parentheses match, pop it from the stack. 

#### **Code Explanation**
1. Create dictionary `par` and map closed parentheses with open parentheses. 
2. Loop through `s` and if current value `i` is an open parenthesis, push it onto the stack.
3. If `i` is a close parenthesis and if stack is empty or the top of stack is not a matching parenthesis with `i`, return `False`
4. Otherwise, pop the top value and return whether the length of stack is 0 or not in Boolean.
