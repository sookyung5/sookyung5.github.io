## **내 풀이**  
--- 

```python
def solution(brown, yellow):
    tot = brown + yellow
    div = []
    for i in range(1,tot+1):
        if tot % i == 0:
            div.append(i)
    
    for w in div:
        h = tot//w
        if w >= h:
            if (w-2) * (h-2) == yellow:
                return [w,h]
```


#### **코드 해설**  
1. 갈색과 노란색을 더해 전체 격자의 합을 구한다.
2. 가로,세로는 총 합의 약수이기 때문에 반복문으로 약수를 구해 리스트에 저장한다.
3. 약수 리스트의 가로가 세로보다 크거나 같고, (가로-2) * (세로-2)값이 yellow와 같으면 (yellow는 전체에서 테두리 제외한 값이므로) [가로, 세로] 형식으로 반환한다.

## **다른 풀이**
---

```python  
def solution(brown, yellow):
    for i in range(1, int(yellow**(1/2))+1):
        if yellow % i == 0:
            if 2*(i + yellow//i) == brown-4:
                return [yellow//i+2, i+2]
```

#### **코드 해설**  
1. 1부터 노란색 영역 넓이의 제곱근까지 반복하며 가능한 세로 길이를 찾는다.
2. yellow의 세로가 i이고 가로가 yellow//i일때, 2×(가로+세로)와 brown-4 (테두리 4개 뺀 값)이 같으면 가로와 세로를 반환한다.
