## **내 풀이**  
--- 

```python
def solution(k, dungeons):
    visited = [0] * len(dungeons)
    
    def dfs(hp, cnt):
        max_dungeons = cnt
        for i in range(len(dungeons)):
            if hp >= dungeons[i][0] and visited[i] == 0:
                visited[i] = 1
                max_dungeons = max(max_dungeons, dfs(hp-dungeons[i][1], cnt+1))
                visited[i] = 0
    
        return max_dungeons
    
    return dfs(k,0)
```
#### **핵심 로직**
사용 알고리즘: 백트래킹, DFS  
1. 모든 던전을 클리어할 수 있는 경우의 수를 보고 피로도에 따라 도중에 탐색을 멈추도록 설계
2. 현재 피로도 >= 최소 필요 피로도 + 방문한 적 없는 던전: 방문처리 
3. 각 조합에서 탐험 던전 수를 비교해 최대값을 선택해 갱신


#### **코드 해설**  
1. 각 던전의 방문여부를 기록할 배열 `visited`를 초기화한다.
2. 던전 탐색 작엄을 처리할 함수 dfs를 정의한다. 매개변수는 피로도 `hp`와 방문 던전 수 `cnt`로 설정했다.
3. 던전 개수만큼 반복하며 현재 피로도가 i번째 던전의 최소 필요 피로도보다 크고 방문한 적이 없는 던전인 경우 방문처리를 한다. 
4. dfs를 재귀적으로 호출하며 현재까지 최대 탐험 가능 던전수와 i번째 던전에서 이동가능한 최대 탐험가능 던전 수 중 최대값을 선택해 갱신한다. 
5. 방문 여부를 0으로 초기화해 백트래킹으로 되돌아갔을 때 방문한 것으로 간주되어 해당 선택지를 고려하지 않는 것을 방지한다. 
6. dfs함수를 호출해 최대 방문 가능한 던전 수를 반환한다.

## **다른 풀이**

```python  
from itertools import permutations

def solution(k, dungeons):
    max_count = 0

    for i in permutations(dungeons, len(dungeons)):
        now_prodo = k
        count = 0 

        for a, b in i:
            if now_prodo >= a:
                now_prodo -= b 
                count += 1
            else:
                break 
        
        max_count = max(max_count, count)

    return max_count


```

#### **코드 해설**  
1. 최대 탐험 던전 수를 0으로 초기화 한다. 
2. 순열을 사용해 던전의 조합을 모두 구한 뒤 각 던전별로 현재 피로도가 최소 필요 피로도보다 큰 경우 피로도를 차감해준 뒤 카운트롤 증가시킨다.
3. 현재 피로도가 최소 필요 피로도보다 적은 경우 반복문을 탈출하고 조합별로 탐험 던전수를 비교해 최대 탐험 던전 수를 반환한다.
